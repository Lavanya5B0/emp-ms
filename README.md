# Employee Management System

A React.js-based Employee Management application that allows you to manage employee data, including adding, editing, deleting, and viewing employees. The app uses **Tailwind CSS** for styling and **React Router** for navigation.

## Features

- Add, edit, and remove employees
- View a list of all employees
- Print employee data
- Profile preview
- Navigate between pages using React Router
- Styled with Tailwind CSS

## Prerequisites

Before running the application, ensure you have the following installed:

- [Node.js](https://nodejs.org/) (v18 or higher recommended)
- [npm](https://www.npmjs.com/) (comes with Node.js)
- Git
